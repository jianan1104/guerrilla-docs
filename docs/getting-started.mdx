# Getting started

Guerrilla is a versatile and automated testing framework tailored for router network testing,  based on the Behavior-Driven Development (BDD) paradigm.
It is built on top of Python and Behave, with support for running on Docker intended for development purposes.



In this document, you'll learn how to install Guerrilla on **Winodws** or **Linux**, set up the enviroment, and develop your first test case.

<hr class="solid" />

## Install Guerrilla on Windows or Linux
Guerrilla can run in either Windows or Linux. We suggest running Guerrilla in Windows for local development
and in Linux for production workloads.
